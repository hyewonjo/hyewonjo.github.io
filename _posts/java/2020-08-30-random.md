# Random
## nextInt 메서드
- int nextInt(int num)
  - 0 부터 num-1까지 범위에 있는 난수를 반환한다.
- int nextInt()
  - 가능한 모든 int 값(양수와 음수)에 걸친 범위의 난수를 반환한다.
